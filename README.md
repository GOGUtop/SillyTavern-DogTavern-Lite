# 🐶 小狗酒馆 Lite · SillyTavern 整合插件

> 从 Android 原生 App 移植的轻量版增强插件  
> **✨ 全平台通用**：iOS Safari / Android / PC 浏览器 均可使用

## 🎯 功能清单

| 功能 | 说明 |
|------|------|
| 🐾 悬浮工具菜单 | 右侧肉掌按钮，支持上下拖动 |
| 🔖 选中生成卡片 | 选中 AI 消息文字即可浮现按钮 |
| 🎨 6 种精美风格 | 暮光紫 / 樱花粉 / 暖茶棕 / 月光蓝 / 晚霞橙 / 墨韵青 |
| 🔊 AI 回复提示音 | Web Audio 合成叮咚音（iOS 也能响） |
| 🧠 智能截断检测 | 回复完成 / 被截断 / 空回三种 Toast 提示 |
| 🎵 一键测试铃声 | 菜单内直接试听 |

## 📥 安装方法

1. 打开 SillyTavern → 右上角 **扩展 (Extensions)** 图标
2. 选择 **"安装扩展 (Install Extension)"**
3. 粘贴本仓库的 URL：  
   `https://github.com/你的用户名/SillyTavern-DogTavern-Lite`
4. 点击安装 → 刷新页面即可看到右侧肉掌 🐾

## 🎬 使用方式

- 点击右侧 🐾 展开菜单
- 选中任意 AI 消息中的文字，会浮现 **🔖 生成卡片** 按钮
- 选择喜欢的风格 → 自动生成精美卡片并下载

## ⚠️ 与完整版 Android App 的差异

本插件为**纯 Web 版**，以下功能为原生 App 独占：
- 💾 资源缓存加速器（WebView 级拦截）
- 📸 长截图拼接
- 🌐 ML Kit 离线翻译错误
- 📱 画中画模式
- 🪟 后台悬浮窗通知
