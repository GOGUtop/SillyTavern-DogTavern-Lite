# 🐶🦴TOP · SillyTavern 增强插件

> 轻量跨平台增强工具  
> **✨ 全平台通用**：iOS Safari / Android / PC 浏览器 均可使用

## 🎯 功能清单

| 功能 | 说明 |
|------|------|
| 🐾 悬浮工具菜单 | 可拖动 + 边缘吸附（50%露出）|
| 🎛️ 扩展面板开关 | 在扩展程序列表中可开关悬浮球 |
| 🔖 选中生成卡片 | 选中 AI 消息文字即可浮现按钮 |
| 🎨 6 种精美风格 | 暮光紫 / 樱花粉 / 暖茶棕 / 月光蓝 / 晚霞橙 / 墨韵青 |
| 🔊 AI 回复提示音 | Web Audio 合成叮咚音（iOS 也能响） |
| 🧠 智能截断检测 | 回复完成 / 被截断 / 空回三种 Toast 提示 |
| 🩺 错误码翻译 | 40+ 内置错误字典 + 微软机翻 |
| 🌐 划词翻译 | 微软 Edge 翻译引擎 |
| 🎵 一键测试铃声 | 菜单内直接试听 |

## 📥 安装方法

1. 打开 SillyTavern → 右上角 **扩展 (Extensions)** 图标
2. 选择 **"安装扩展 (Install Extension)"**
3. 粘贴本仓库的 URL：  
   `https://github.com/你的用户名/SillyTavern-DogTop`
4. 点击安装 → 刷新页面
5. 在 **扩展程序** 面板中可以看到 **🐶🦴TOP** 的设置项

## 🎬 使用方式

- 在 **扩展程序面板** 中找到 🐶🦴TOP，展开可以控制：
  - 🐾 悬浮球 开/关
  - 🔊 提示音 开/关
  - 🌐 划词翻译 开/关
- 点击屏幕边缘的 🐾 悬浮球展开菜单
- 悬浮球支持 **拖动**，松手后自动 **吸附到最近边缘**（露出50%）
- 选中任意 AI 消息中的文字，会浮现 **🔖 生成卡片** 按钮

## 💡 关于长截图

由于 iOS Safari 对大尺寸 Canvas 有严格内存限制，
长截图功能在苹果设备上无法稳定工作，本版本已移除。
